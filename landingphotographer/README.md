# White Rose Studio - Wedding Photography Landing Page

## Setup Instructions

1. **WhatsApp Configuration**
   - Open `index.php`
   - Update line 3: Replace `60123456789` with your actual WhatsApp number
   - Format: Country code + number (no spaces or special characters)

2. **Logo Setup**
   - Place your logo image in the `img/` folder
   - Name it `logo.png` (or update the filename in index.php)
   - Recommended size: 200x50px (transparent background)

3. **Run the Website**
   - Place all files in your web server directory (htdocs/www)
   - Access via: `http://localhost/your-folder-name`
   - Or use PHP built-in server: `php -S localhost:8000`

## Customization

- **Colors**: Edit CSS variables in `style.css` (lines 7-12)
- **Content**: Edit text directly in `index.php`
- **Images**: Replace Unsplash URLs with your own images
- **Contact Info**: Update footer section in `index.php`
